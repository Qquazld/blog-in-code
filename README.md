# Blog in Code

This project was completed as part of "The Complete 2024 Web Development Bootcamp" by Angela Yu on Udemy.

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- EJS

## Installation

Clone this repo:

```shell
git clone https://github.com/Qquazld/Forkify-yann.git
```

```shell
cd blog-in-code
```

Install dependencies:

```shell
npm install
```

Run the app:

```shell
npm start
```
